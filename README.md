# Decibel Meter

コンサートホールのスクリーン向け、リアルタイム音量表示アプリケーション。

マイク入力から dB SPL を計算して大型表示し、120秒間の音量推移をグラフで可視化します。

## 動作環境

- Windows 10 / 11
- Python 3.10 以上
- マイク入力デバイス

## インストール

```bash
pip install -r requirements.txt
```

## 起動

```bash
python decibel_meter.py
```

## 起動フロー

```
起動
 ├─ calibration.json あり → [startup] 前回値確認
 │    ├─ Enter  → [noise_setup] 暗騒音設定
 │    ├─ S      → [main] スキップ
 │    └─ R      → [calib_step1]
 └─ なし → [calib_step1] キャリブレーション STEP 1
               ↓ Enter
           [calib_step2] キャリブレーション STEP 2
               ↓ Enter（または S で1点のみ）
           [calib_confirm] 確認
               ↓ Enter / S
           [noise_setup] 暗騒音設定
               ↓ Enter
           [noise_measure] 計測中（自動完了）
               ↓
           [main] 計測画面
```

## キャリブレーション

dB SPL の絶対値表示には1点または2点キャリブレーションが必要です。

1. スマホの騒音計アプリ（例: [NIOSH SLM](https://www.cdc.gov/niosh/topics/noise/app.html)）を用意する
2. **STEP 1**: 静かな音（pp の弱音など）を出し、騒音計の値を入力して Enter
3. **STEP 2**: 大きな音（ff の強奏など）を出し、騒音計の値を入力して Enter（`S` で1点のみ確定）
4. キャリブレーション結果は `calibration.json` に保存され、次回起動時に自動読み込み

> スキップした場合は相対値で表示されます（絶対値は不正確）。

## 暗騒音ゲート

キャリブレーション後に暗騒音（ホール静寂時の床雑音）を測定し、それ以下の音量では表示を凍結するノイズゲートを設定できます。

| 設定項目 | 既定値 | 範囲 | 説明 |
|----------|--------|------|------|
| 計測時間 | 10秒 | 3〜60秒 | 暗騒音サンプリング時間 |
| パーセンタイル | 10% | 5〜50% | 下位N%をフロア値とする |
| マージン | 5 dB | 0〜20 dB | フロアに加算するゲート閾値 |

`noise_setup` 画面で ↑↓ キーで行選択、←→ キーで値を調整します。`S` でスキップ（ゲートなし）。

## 操作方法

### キャリブレーション・設定画面

| キー | 動作 |
|------|------|
| `Enter` | 確定 / 次へ進む |
| `S` | スキップ |
| `R` | キャリブレーションからやり直し |
| `↑` / `↓` | 項目選択（noise_setup） |
| `←` / `→` | 値変更（noise_setup） |
| `Q` / `Esc` | 終了 |

### 計測画面（main）

| キー | 動作 |
|------|------|
| `Space` | 計測開始 / 停止 |
| `N` | 暗騒音を再測定 |
| `F` | フルスクリーン切替（プロジェクター投影時） |
| `Tab` | オペレーター情報オーバーレイ表示/非表示 |
| `R` | 再キャリブレーション |
| `Q` / `Esc` | 終了 |

## 表示について

- **数字の色**: 音量レベルに応じて変化
  - 緑: ～70 dB（静かな会話レベル）
  - 黄: 70～90 dB（交通騒音レベル）
  - 橙: 90～110 dB（工事現場レベル）
  - 赤: 110 dB～（ライブ・航空機エンジンレベル）
- **グラフ**: 直近120秒間の音量推移（12.5 Hz サンプリング）
- **ノイズゲート発動時**: 数字が暗くなり表示が凍結

## 技術仕様

| 項目 | 値 |
|------|----|
| サンプリングレート | 44,100 Hz |
| ブロックサイズ | 1,024 サンプル（約23ms） |
| 平滑化 | 指数移動平均（α=0.3） |
| グラフ更新レート | 12.5 Hz |
| グラフ履歴 | 120秒 |
| キャリブレーション方式 | 1点または2点線形回帰（dB_SPL = a × dB_raw + b） |
| オーディオ入力 | WASAPI 排他モード優先、失敗時は共有モードへ自動フォールバック |
| ウィンドウ | シングルウィンドウ、リサイズ可能 |
