# Decibel Meter

コンサートホールのスクリーン向け、リアルタイム音量表示アプリケーション。

マイク入力から dB SPL を計算して大型表示し、120秒間の音量推移をグラフで可視化します。

## スクリーンショット

キャリブレーション画面 → 計測画面（大型dB表示 + 折れ線グラフ）の2ステップで使用できます。

## 動作環境

- Windows 10 / 11
- Python 3.10 以上
- マイク入力デバイス

## インストール

```bash
pip install -r requirements.txt
```

## 起動

```bash
python decibel_meter.py
```

## キャリブレーション

dB SPL の絶対値表示には1点キャリブレーションが必要です。

1. スマホの騒音計アプリ（例: [NIOSH SLM](https://www.cdc.gov/niosh/topics/noise/app.html)）を用意する
2. 安定した音（掃除機、ピンクノイズなど）を出す
3. STEP 1: 静かめの音で騒音計の値を入力
4. STEP 2: 大きめの音で騒音計の値を入力（2点キャリブレーションで精度向上）
5. キャリブレーション結果は `calibration.json` に保存され、次回起動時に自動読み込み

> スキップした場合は相対値で表示されます（絶対値は不正確）。

## 操作方法

| キー | 動作 |
|------|------|
| `Space` | 計測開始 / 停止 |
| `F` | フルスクリーン切替（プロジェクター投影時） |
| `Tab` | オペレーター情報オーバーレイ表示/非表示 |
| `R` | リセット（キャリブレーションからやり直し） |
| `Q` / `Esc` | 終了 |

## 表示について

- **数字の色**: 音量レベルに応じて変化
  - 緑: ～70 dB（静かな会話レベル）
  - 黄: 70～90 dB（交通騒音レベル）
  - 橙: 90～110 dB（工事現場レベル）
  - 赤: 110 dB～（ライブ・航空機エンジンレベル）
- **グラフ**: 直近120秒間の音量推移

## 技術仕様

| 項目 | 値 |
|------|----|
| サンプリングレート | 44,100 Hz |
| ブロックサイズ | 1,024 サンプル（約23ms） |
| 平滑化 | 指数移動平均（α=0.3） |
| グラフ更新レート | 12.5 Hz |
| キャリブレーション方式 | 1点または2点線形回帰（dB_SPL = a × dB_raw + b） |
