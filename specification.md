# Specification

## アプリケーション概要

マイク入力された音量をリアルタイムでデシベル表示するコンサートホール向けアプリケーション。

---

## UI

- **Borderless Fullscreen**
- 画面上部：現在の音量を特大文字で表示（1〜3桁）＋ `dB`
- 画面下部：音量の変遷を折れ線グラフで表示（直近120秒）

---

## 操作

| キー | 動作 |
|------|------|
| `Space` | 計測開始 / 停止 |
| `R` | 初期画面へリセット（キャリブレーションからやり直し） |
| `F` | フルスクリーン切替 |
| `Tab` | オペレーター情報オーバーレイ表示 / 非表示 |
| `Q` / `Esc` | 終了 |

---

## オプション（後続実装候補）

1. 通算での音量の最大・最小値を表示
2. 現在の音量の騒音レベル表示（例：120dB越えはジェット機のアイコンなど）

---

## 実行環境

- Windows 10 / 11
- Python 3.10 以上

---

## 使用する企画

> オーケストラが出す音量はジェット機並みの120dBに達する一方、呼吸音すら騒音になりそうな静寂な音も出せる…生演奏の大きな魅力である音量の幅を味わってみましょう！

**コンサートの流れ**

1. チューニングの A を弦で最小音量〜全員で最大音量までボリュームアップ
2. 《火の鳥》フィナーレを演奏
3. コンサートホールのスクリーンに現在の音量をリアルタイム表示し、聴衆と共有
